# LionDisplays
allows the Server to place visual elements like Items, Textures, Text, Rectangles and more on your Screen

# Dependencies
This mod does depend on
- FabricLanguageKotlin

And I recommend installing it along with
- ClothConfig
- ModMenu

# Special Features
- Compass: The Compass shows you excactly in which direction to walk.
- Configuration: You can configure default values, disable the elements on screen and more.
- Transition: When used with LionVelocity, this mod shows a clean fade to black when during reconfiguring and World Loading time.
